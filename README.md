# 🦐 癒し水槽 ― 海老育成ゲーム

ブラウザで動くリアルな海老育成シミュレーションゲームです。  
Canvas 2D で描画された美しい水槽の中で、ビーシュリンプとクリアシュリンプを育てましょう。

🔗 **プレイはこちら → [https://junia2009.github.io/my_aquarium/](https://junia2009.github.io/my_aquarium/)**

## ✨ 特徴

- **単一HTMLファイル** — 外部ライブラリ不要、index.html だけで完結
- **リアルな水中表現** — Beer-Lambert 法による光の減衰、コースティクス、ゴッドレイ、パーリンノイズの砂底
- **解剖学的に正確なエビ** — ベジエ曲線で描画されたビーシュリンプ＆クリアシュリンプ
- **本格的な育成システム** — 空腹・HP・水温・水質・病気・繁殖をシミュレート
- **レスポンシブ対応** — スマートフォン（縦向き）、タブレット、デスクトップに対応
- **PWA 対応** — ホーム画面に追加してアプリのように使える
- **BGM** — Web Audio API による環境音楽（4レイヤー合成）

## 🎮 操作方法

| ボタン | 機能 |
|--------|------|
| 🍤 **餌** | 通常フードを投下（満腹度 +15） |
| 🥬 **野菜** | 野菜を投下（満腹度 +10、腐りにくい、水質に優しい） |
| 💧 **水換え** | 水質 +30、コケも一部除去 |
| 🌡️ **ヒーター** | ON/OFF 切替。ONで水温を25℃に維持 |
| ⚙️ **ろ過** | ON/OFF 切替。ONで水質の自然浄化をサポート |
| 🔇 **BGM** | 環境音楽のON/OFF |

## 🐛 ゲームシステム

### 生命管理
| 要素 | 説明 |
|------|------|
| **空腹度** | 時間経過で減少。20以下でHP低下。餌を食べて回復 |
| **HP** | 0でエビが死亡（浮上＋フェードアウト）。水質・水温・空腹で影響 |
| **脱皮** | 定期的に発生。水質・水温が悪いと **脱皮失敗** のリスク（HP大幅減少） |
| **抱卵・孵化** | 条件が揃うとメスが抱卵 → 60〜100秒後に2〜4匹の稚エビが誕生 |
| **病気** | 水質・水温悪化で発症リスク。近くのエビに**伝染**。60〜120秒で自然回復 |

### 水質管理
| 要因 | 水質への影響 |
|------|-------------|
| 自然劣化 | -0.025/秒（ベース） |
| バイオロード | -0.003/秒/匹 |
| 残餌腐敗 | 30秒放置で腐敗開始 → -0.012/個/秒 |
| 脱皮殻 | -0.006/殻/秒 |
| 死体 | 即時 -8 |
| 排泄 | 食後15〜30秒後に -0.3 |
| 水草浄化 | +0.0008/本/秒 |
| ろ過装置 | +0.012/秒（ON時） |

### 環境システム
| 要素 | 説明 |
|------|------|
| **水温** | 季節に連動（1月=8℃〜7月=32℃）。適温は22〜26℃ |
| **ヒーター** | ONで25℃に自動維持 |
| **照明サイクル** | JST 7〜19時は昼間 → コケが成長 |
| **コケ** | 照明で成長し、ガラスに緑の苔が付着。水換えで一部除去。エビが自然に食べることも |
| **隠れ家** | 流木風シェルター2つが自動配置。過密ストレスを軽減＋微量HP回復 |
| **過密ストレス** | 8匹以上でHP低下（隠れ家で軽減可能） |

## 🦐 エビの種類

| 種類 | 特徴 |
|------|------|
| **ビーシュリンプ** | 赤白バンド模様。やや機敏。初期3匹 |
| **クリアシュリンプ** | 透明ボディ。他のエビについて行くことがある。初期2匹 |

### エビの行動
- **ツマツマ** — 前脚で地面をつつく
- **歩行** — 水草の近くを散策
- **遊泳** — 水中を泳ぎ回る
- **浮遊** — ゆったり漂う
- **食事** — 落ちた餌に向かって移動して食べる
- **追従** — 他のエビの後を追う（クリアシュリンプ）
